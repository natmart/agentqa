## Scoring Rubric (Internal)

**Ideal beta candidate:**
- 20-50 engineers (+2 points)
- TypeScript/JavaScript primary (+2 points)
- Uses AI coding tools daily (+2 points)
- No dedicated QA team (+2 points)
- GitHub for code hosting (+1 point)
- 15+ PRs per week (+1 point)
- Available for onboarding call (+1 point)
- Willing to share weekly feedback (+1 point)

**Score 8+:** Priority invite
**Score 5-7:** Good fit, invite if capacity
**Score <5:** Waitlist
